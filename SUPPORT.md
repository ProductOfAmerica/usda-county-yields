# Support

Use GitHub Issues for support:

- Data coverage questions: include the state, county, crop, year, and the URL you queried.
- Bug reports: include the expected value or behavior, actual response, and whether you used `@main` or a pinned commit SHA.
- Feature requests: describe the crop, geography, or workflow that needs support.

For point-in-time consistency, use jsDelivr URLs pinned to a commit SHA instead of `@main`.
